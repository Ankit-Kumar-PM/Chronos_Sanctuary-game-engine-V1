## Overview

**Chronos Sanctuary** is a puzzle-driven terminal game where the player wakes up trapped inside a mysterious hexagonal chamber. To escape, they must explore all four cardinal directions, collect clues, solve riddles, repair an antique watch, and uncover the final six-digit code that unlocks the vault door.

The game combines:

* Interactive storytelling
* Inventory management
* Puzzle solving
* Exploration mechanics
* Multiple clue chains
* Escape room gameplay

---

## Story

You awaken on a cold stone floor inside a strange black chamber.

A synthetic voice echoes through hidden speakers:

   "Look North, East, South, and West. Gather the clues. Connect the pieces."

A massive vault door blocks the only exit.

Somewhere inside the room lies the code needed to escape.

Can you solve the mystery of the Chronos Sanctuary?

## Features

### Exploration System

* North Wall
* South Wall
* East Wall
* West Wall

Each wall contains unique clues and interactive objects.

### Inventory System

Collect and manage important items such as:

* Broken Pocket Watch
* Watch Hour Hand
* Repaired Pocket Watch

### Puzzle Mechanics

* Riddle-based password puzzles
* Hidden clue discovery
* Watch repair mechanic
* Time-setting puzzle
* Final vault code challenge

### Typewriter Effect

Immersive dialogue and narration using animated text output.

### State Management

The game tracks:

* Visited locations
* Inventory items
* Puzzle progress
* Door unlock conditions

## Game Flow

1. Explore the North Wall.
2. Obtain the broken pocket watch.
3. Explore the South Wall.
4. Solve the wooden box riddle.
5. Obtain the watch hand.
6. Repair the watch.
7. Discover the hidden watch code.
8. Explore the East Wall.
9. Solve the painting clue.
10. Obtain the digital code.
11. Combine clues.
12. Unlock the vault door.
13. Escape.
